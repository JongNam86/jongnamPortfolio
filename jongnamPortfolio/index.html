TECH SKILLS

다양한 언어 습득 및 프로그래밍 툴을 다루어보았습니다.




  • JAVA

    JAVA는 처음 접하게 된 프로그래밍 언어로, 변수 선언부터 반복문, 객체, 클래스, 상속 등의 기본기를 익힐 수 있었습니다. 이를 기반으로 와인 주문시스템 서비스를 구현했습니다.


  • HTML / JavaScript / CSS

    JavaScript의 기본 문법을 익히고, jquery를 통해 직접 웹프론트에서 구동하도록 구현해 보았습니다. 그리고 HTML과 CSS를 이용하여 HTML 템플릿을 구성하고, 모바일에서도 작동 가능하도록 반응형 페이지를 구현했습니다.

  • MySQL / MariaDB

    데이터베이스로는 관계형 데이터베이스인 MySQL과 MariaDB를 사용했습니다. Eclipse에서는 JDBC 드라이버를 이용하여 연동하는 작업을 진행했고, 종합프로젝트에서는 MariaDB를 이용하여 작업을 진행했습니다.

  • JSP

    JSP는 HTML 코드에 JAVA 코드를 넣어 동적 웹페이지를 생성하는 웹 애플리케이션 도구로, 배움을 통해 회원가입 & 로그인 기능 등 다양한 기능을 구현해 보았습니다.



  • Eclipse / IntelliJ

    Eclipse를 이용하여 JAVA 및 JSP를 공부하며 다양한 기능 구현을 해보았고, IntelliJ를 이용하여 지금까지 배운 HTML, JavaScript 등의 언어를 다양하게 사용하여 종합 프로젝트인 부산 맛집 소개 웹사이트를 구현했습니다.


  • Apache Tomcat

    Apache Tomcat은 아파치 소프트웨어 재단에서 개발한 서블릿 컨테이너만 있는 웹 애플리케이션 서버이며, Tomcat과 Eclipse의 연동을 통해 본인이 작업한 개인프로젝트를 웹페이지에서 볼 수 있도록 구현했습니다.



  • SpringBoot

    SpringBoot는 가장 대중적인 응용프로그램 개발 프레임워크로, 간단한 어노테이션 및 프로퍼티 설정으로 원하는 기능을 빠르게 적용할 수 있어서, Eclipse를 사용할 때보다 편리함을 느낄 수 있었습니다.



  • GitHub

    GitHub를 이용하여 대부분의 개인 & 팀 프로젝트의 소스 코드를 관리하고 있으며, GitHub 서버를 활용하여 개인 포트폴리오 사이트를 관리하고 있습니다.




  • Amazon Web Services

    종합프로젝트 과정에서 Amazon Web Services의 프리티어 계정 가입을 통하여 EC2를 확장하고, AWS를 서버로 활용하여 마지막 종합프로젝트를 관리하고 있습니다.

프로젝트를 진행할 때 가장 기본이 되는 추진체계 & 프로젝트 기획서 부분입니다.
프로젝트 일정 및 주제 선정 등 많은 요소를 결정하는 중요한 단계입니다.

< 클래스 다이어그램 >


< BmjApplication > < Entity 클래스 > < Controller 클래스 > < Service 클래스 > < DTO 클래스 > < SecurityConfig 클래스 > < Repository 클래스 > < 유스 케이스 다이어그램 >

클래스 다이어그램은 클래스의 구성요소 및 클래스 간의 관계를 표현하는 대표적인 UML입니다. 이를 통해 시스템의 일부 또는 전체의 구조를 나타낼 수 있고, 유스 케이스 다이어그램은 사용자, 그리고 사용자가 수반한 다른 유스 케이스 간의 관계를 보여주는 사용자-시스템 간 상호작용의 표현입니다. 유스 케이스 다이어그램은 각기 다른 시스템 사용자와 각기 다른 유스 케이스를 식별할 수 있으며 다른 유형의 다이어그램이 수반되기도 합니다.

< 메인 페이지 >

주요 기능은 회원가입 및 로그인, 검색 기능, 동네별 맛집 리스트로 구성되어있습니다.

< EAT딜 페이지 >



< 비회원일 경우 > < 회원일 경우 >

EAT딜 페이지에서는 저희 BMJ WHERE과 제휴 맺은 가게의 정보 및 할인 쿠폰을 제공합니다.
회원만 쿠폰을 받을 수 있으며, 비회원일 경우 회원가입을 통해 할인 쿠폰을 받을 수 있습니다.

< 카테고리별 맛집 리스트 >



카테고리별 맛집 리스트 페이지에서는 한식, 일식, 중식, 양식, 카페 5가지 항목을 보여줍니다.
주요 기능은 카테고리별 TOP3 음식점을 추천해주고 있으며, 아래쪽에는 카테고리별 맛집 리스트 목록을 보여줍니다.

< 맛집 리스트 >




< 관리자모드로 로그인 > < 새로운 맛집 등록 버튼 생성 > < 맛집 등록 페이지 >

맛집 리스트 페이지에서는 데이터베이스에 저장된 모든 맛집 목록을 보여줍니다.
주요 기능은 관리자모드로 로그인하여 맛집을 등록하는 기능과 다양한 방식의 검색기능이 있습니다.

< 맛집 조회 >



< 댓글 등록 기능 > < 댓글 등록 후 >

맛집 조회 페이지에서는 본인이 클릭한 맛집의 가게명, 연락처 등 많은 정보를 볼 수 있습니다.
주요 기능은 로그인 시 댓글 작성이 가능하며, 작성 후 댓글 정보가 자동으로 출력되는 것을 볼 수 있습니다.
